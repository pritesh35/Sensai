## Summary of Commands
For quick reference, here are all the commands you need to run in order:

# 1. Clone the repo
```
git clone https://github.com/piyush-eon/ai-career-coach.git
```
# 2. Enter the directory
```
cd ai-career-coach
```
# 3. Install dependencies
```
pnpm install
```
# (Create your .env file here)

# 4. Sync the database
```
pnpm prisma db push
```
# 5. Run the app
```
pnpm dev
```
